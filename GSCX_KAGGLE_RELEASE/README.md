# GSC-X / GDS Public Research Release

This package contains selected public benchmark and cosmology outputs from the GSC-X and GDS research framework.

## Included Sections

### control/

Fusion-core nonlinear endurance benchmark results:

- TRUE N3000 confirmation benchmark
- 9000 total simulations
- delayed noisy nonlinear regime
- comparison against classical PID control

Key result:

GSCX_V91 achieved:
- 100% survival
- 0.0 unsafe time
- substantially lower instability amplitude
- ~617x higher yield-per-energy than PID

### cosmology/

Selected outputs from the GDS cosmic-fate numerical scans:

- stable asymptotic expansion
- no Big Rip in the clean scan
- no global collapse detected
- future convergence toward q -> -1

The cosmology section also includes visual comparisons with:
- ΛCDM
- Big Rip scenarios
- vacuum decay concepts
- alternative cosmic-fate models
